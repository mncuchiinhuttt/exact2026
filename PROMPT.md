### GPU Backend Detection
  At startup, detect the available GPU backend and configure accordingly:

  ```python
  import torch

  def detect_backend() -> str:
      """Detect available GPU backend: 'cuda', 'rocm', or 'cpu'."""
      if torch.cuda.is_available():
          # Check if this is ROCm (AMD) or CUDA (NVIDIA)
          if torch.version.hip is not None:
              return "rocm"
          return "cuda"
      return "cpu"

  BACKEND = detect_backend()
  ```

  Use BACKEND to decide quantization strategy:
  - "cuda": use BitsAndBytesConfig (4-bit NF4) + prepare_model_for_kbit_training
  - "rocm": load full bf16, no quantization (bitsandbytes does not support ROCm;

  - "cpu": load full fp32, warn user

## B2. answer_parser.py

Implement parse_answer(text: str, reasoning: str = None) -> dict

Strategy (in order):
  1. Search in text for line matching: r"ANSWER:\\s*([\\d\\.\\+\\-eE×x]+)\\s*([A-Za-zΩμ°²³/·\\-]+)?"
  2. If not found, search last 500 chars of text for same pattern
  3. If not found and reasoning is not None, search last 500 chars of reasoning
  4. If not found, search for any standalone number in last 3 lines of text

Return:
  {
    "value_raw": str,         # matched string for value
    "unit_raw": str,          # matched string for unit (or "")
    "value": float or None,   # parsed float
    "unit": str,              # normalized unit from unit_normalizer
    "value_si": float,        # value after prefix expansion
    "source": "answer_line" | "text_tail" | "reasoning_tail" | "last_number" | "failed"
  }

## B3. voting.py

Implement vote(parsed_answers: list[dict], tolerance: float = 0.03) -> dict

Algorithm:
  1. Filter out records where value is None
  2. Group by normalized_unit
  3. Within each unit group, cluster numeric values by relative tolerance:
     two values a, b are in the same cluster if abs(a-b)/max(abs(a),abs(b),1e-12) <= tolerance
  4. Pick the cluster with most members (ties: pick cluster with smallest std)
  5. Within winning cluster, pick the median value
  6. Return:
     {
       "answer": float,
       "unit": str,
       "confidence": winning_count / total_non_null,
       "winning_count": int,
       "total": int,
       "all_clusters": list   # for debugging
     }

## B8. pipeline_physics.py

Implement run_physics(question: str, config: dict = None) -> dict

config defaults:
  k = 5            # self-consistency samples
  mode = "full"    # prompt mode
  fast = False     # if True: k=1, mode=compact, skip direct solver
  direct_first = True

Flow:
  1. domain = classify_domain(question)
  2. If direct_first: result = try_direct_solve(question, domain)
     If result and fast: return format_output(result, source="direct_solver")
  3. formulas = retrieve_formulas(domain, question)
  4. messages = build_physics_prompt(question, formulas, mode=config["mode"])
  5. generations = client.generate_k(messages, k=config["k"])
  6. parsed = [parse_answer(g["content"], g["reasoning"]) for g in generations]
  7. vote_result = vote(parsed)
  8. If direct_solver result exists and units match and within 5%: use direct answer, source="direct_refinement"
  9. winning_gen = first generation whose answer matched vote_result["answer"]
  10. Return:
      {
        "answer": vote_result["answer"],
        "unit": vote_result["unit"],
        "confidence": vote_result["confidence"],
        "explanation": extract_explanation_block(winning_gen["content"]),
        "cot": extract_cot_block(winning_gen["content"]),
        "raw_think": winning_gen["reasoning"],
        "domain": domain,
        "formulas_used": [f["id"] for f in formulas],
        "source": source_tag,
        "all_answers": parsed
      }

  - extract_explanation_block: search for "EXPLANATION:" section → return text

## B10. router.py

Implement route_and_run(input_data: dict, config: dict = None) -> dict

  subtask = classify_subtask(input_data)

  if subtask == 2:
    question = input_data.get("question", "")
    return run_physics(question, config)

  elif subtask == 1:
    # Handle multiple questions in one record
    premises = input_data.get("premises-NL") or input_data.get("premises_nl") or []
    questions = input_data.get("questions") or []

    # If single question string (not list), wrap it
    if isinstance(questions, str):
        questions = [{"question": questions}]

    results = []
    for q_item in questions:
        q_text = q_item.get("question") if isinstance(q_item, dict) else str(q_item)
        result = run_edu(premises, q_text, config)
        results.append(result)

    # Return single result if one question, list if multiple
    return results[0] if len(results) == 1 else results

  else:
    return {"error": "Cannot determine subtask from input shape", "input_keys": list(input_data.keys())}
